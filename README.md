# EbooksPortal


### Clone Project
```
 git clone https://github.com/Ajay2810-hub/EbooksPortal.git
```

### Install Requirements
```
 pip install -r requirements.txt 
```

### Create database and user with following credentials
|   Database Name   |   EbooksPortal   |
| ----------------- | ---------------- |
|     ` User `      | ` EbooksPortal ` |
|   ` Password `    |   ` portal@2 `   |
